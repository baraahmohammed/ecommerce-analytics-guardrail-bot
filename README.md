

#  E-Commerce Analytics & Guardrail Chatbot

An enterprise-grade, production-ready intelligent analytics assistant powered by advanced language models, structured data processing layers, and robust guardrails to prevent hallucination and secure data insights. Built with a sleek **Streamlit** user interface.

##  Features
- **Data Ingestion Layer**: Optimized streaming and caching of multi-dimensional e-commerce datasets (`DatasetSheet1.csv`) using `st.cache_data`.
- **Analytical Logic Core**: Sophisticated processing of sales, customer behaviors, metrics, and inventory trends.
- **Safety & Guardrail Layer**: Strict prompt boundaries and response-validation logic to guarantee data accuracy and mitigate safety risks.
- **Interactive Dashboard UI**: Beautiful, responsive conversational interface built entirely on Streamlit.

##  Tech Stack & Tools
- **Frontend/UI**: Streamlit
- **Data Processing**: Pandas, NumPy
- **Version Control & Hosting**: Git, GitHub

##  Getting Started

### 1. Prerequisites
Make sure you have Python 3.9+ installed on your system.

### 2. Installation & Setup
Clone the repository and install the required dependencies:
```bash
git clone [https://github.com/baraahmohammed/ecommerce-analytics-guardrail-bot.git](https://github.com/baraahmohammed/ecommerce-analytics-guardrail-bot.git)
cd ecommerce-analytics-guardrail-bot
pip install -r requirements.txt
